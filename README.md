# OpenAPI Automatic API Tester

This project downloads a Swagger/OpenAPI v2 specification and generates Jest tests to validate endpoints, responses, and schemas.

Quick start

1. Install dependencies:

```bash
npm install
```

2. Refresh the Swagger spec (downloads to `src/swagger.json`):

```bash
npm run swagger:refresh
```

3. Generate tests and run them:

```bash
npm run test:generated
```

Reports

- JSON report: `reports/results.json` (when running `npm run report`)
- HTML report: `reports/report.html` (generated by Jest HTML reporter)

Configuration

Copy `.env.example` to `.env` to override `BASE_URL`, `TIMEOUT`, `RETRIES`, and `LOGGING`.

Files

- `src/swagger-loader.js` — downloads the Swagger JSON.
- `src/api-client.js` — axios wrapper with retries and logging.
- `src/test-generator.js` — generates `tests/generated.test.js` from the spec.
- `src/utils/validator.js` — Ajv schema validator.
- `tests/generated.test.js` — example generated tests.
